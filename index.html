dialog

dialog 是一个豆瓣风格的 jQuery 对话框，旨在提供一个轻量、简洁的原生对话框替换工具。

演示 Demo

dialog.ok(); dialog.ok(); dialog.ok(); dialog.ok(); dialog.ok(); dialog.ok(); dialog.ok();

载入 Init

<link rel="stylesheet" type="text/css" href="css/dialog.css" />
<script src="js/jquery.js"></script>
<script src="js/dialog.js"></script>
